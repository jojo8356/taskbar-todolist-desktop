# Source Layout

This folder contains the Vanilla TypeScript frontend generated by the official Tauri starter.

Current scaffold:

- `main.ts`: starter frontend entrypoint.
- `styles.css`: starter frontend styles.
- `assets/`: starter static assets.

Later stories will add tray lifecycle, compact panel UI, and Rust-side local persistence. Phase 1 remains desktop-local and does not include backend, sync, mobile framework, cloud, account, or Node ORM code.
